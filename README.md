# Sports History Hub

Sports History Hub was created as a final project for CS/LIS 472. The purpose of the website is to teach users about sports history through interactive pages, quizzes, and games.

Users can explore famous moments in sports, test their knowledge, and learn more about athletes and historical events in an engaging way.

## Features

- Home page with project overview
- Guess Who sports game
- Historic Moments page featuring the Jordan Flu Game
- Soccer ceasefire history page
- Sports quizzes
- Sports Showcase page
- Theme toggle using localStorage
- Responsive layout using Bootstrap

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- GitHub Pages
